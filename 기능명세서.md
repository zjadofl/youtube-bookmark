# 기능 명세서
1. 북마크 등록 ✅
    - 시간 출력 ✅
    - 메모 입력 / 수정 ✅
    - youtube 영상일 때 버튼 활성화 ✅
    - youtube 영상이 아닐 때 안내문구 출력 ✅

        (안내문구 : "Youtube 영상 페이지일 때만 버튼이 활성화됩니다.")
    - youtube 영상이 아닐 때 버튼 비활성화 ✅
2. 북마크 저장
    - 사용자별 북마크 저장
3. 재생바에 북마크 표시
    - 노란색으로 북마크 표시
    - 재생바의 북마크 hover시 메모 표시
    - 북마크 클릭시 이동
4. 북마크 삭제
5. 북마크 리스트
    - 리스트에 저장된 영상 페이지일 때, 관련 북마크 노출
    - youtube 영상 페이지가 아닐 경우, 북마크 영상 리스트 노출
6. 태그 관리
    - 태그 추가
    - 태그 수정
    - 태그 삭제
7. 북마크 검색
    - 태그로 필터링
    - 검색어로 필터링
8. 북마크 공유 ✅
    - 북마크 시간의 주소를 공유 ✅
